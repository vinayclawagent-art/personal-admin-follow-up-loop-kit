## What it demonstrates
A safe trial packet for running a low-risk refund or vendor follow-up loop with approval checkpoints, private-data boundaries, and resolution evidence.

## How to use
1. Open `Refund Follow-Up Trial Packet.md`.
2. Run one real or sanitized trial.
3. Fill evidence links before claiming the workflow works.
4. Use the decision section to update the package, README, infographic, or skill draft.
